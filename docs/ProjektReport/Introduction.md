# Introduction

## General

In the Mechatronics degree program at Karlsruhe University of Applied Sciences
(HKA), students are required to carry out a joint project under the supervision
of a professor. Project topics may be proposed by students themselves or offered
by professors. The present project was initiated and supervised by Prof. Dr.
Michael Hoffmeister.

## Topic

The objective of this project is to establish an initial bridge between Asset
Administration Shells (AAS) and CAD systems. An AAS serves as a digital
container for all data accumulated throughout a product's lifecycle. This
project acts as a pilot to demonstrate how integrating AAS with CAD systems can
streamline modern workflows and daily operations. To make the process more
tangible, the first step involved defining specific use cases that leverage
this connection. Subsequently, the project focused on developing code to
implement two of these use cases, which will be showcased as a live demo during
the final presentation.
