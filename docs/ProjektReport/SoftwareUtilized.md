## Software Utilized

The project combines proprietary CAD software with open source tooling and
libraries. The main components and their licenses are summarized below.

- Creo 12 (PTC): Proprietary CAD system used under a student license. Use is
  governed by the PTC software license agreement; redistribution is not
  permitted outside that agreement.
- PyCharm: Development environment by JetBrains. The Community Edition terms
  cover Apache 2.0 open source code plus bundled proprietary plugins; use of
  free editions for commercial development is permitted. Pro features require a
  paid subscription after the trial period.
- Python: PSF License (v2). Permissive use, modification, and redistribution,
  provided the PSF license text and copyright notice are retained in
  distributions.
- pytest: MIT License. Permissive use, modification, and redistribution,
  provided the copyright and license notice are included in distributions.
- Creoson Server (Simplified Logic): MIT License. Permissive use, modification,
  and redistribution, provided the copyright and license notice are included in
  distributions.
- Creopyson (Python client for Creoson): MIT License. Permissive use,
  modification, and redistribution, provided the copyright and license notice
  are included in distributions.
- Eclipse BaSyx Python SDK (`basyx_python_sdk`): MIT License. Permissive use,
  modification, and redistribution, provided the copyright and license notice
  are included in distributions.
- PyECMA376-2 (`pyecma376_2`): Apache License 2.0. Permissive use, modification,
  and redistribution with required license text and NOTICE retention for
  attribution where applicable.
