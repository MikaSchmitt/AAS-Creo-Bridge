# AAS-Creo-Bridge
Tool for integrating Asset Administration Shells (AAS) with Creo Parametric CAD software.

# Getting Started for Developers

1. Clone the repository 
2. Set up a virtual environment with Python 3.12
3. Run setup.py to install all requirements
    - use `--install-deps` to install all necessary python packages
    - use `--embedded-jre` if you don't have a JRE 21 installed on your system. This will download a version of creoson with an embedded JRE. 
    - use `--force` if you want to reinstall a different version of creoson.
